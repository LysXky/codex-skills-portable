# Agent Skills Catalog

Use this skill to consult the bundled VoltAgent `awesome-agent-skills` catalog without browsing every time.

## Workflow

1. Read or search `references/awesome-agent-skills.md` for the user's domain, framework, language, or agent workflow.
2. Prefer already-installed local Codex skills when they cover the same need.
3. If a catalog entry points to an external repository, treat it as untrusted until reviewed: inspect its `SKILL.md`, scripts, install steps, licenses, and network/credential behavior before recommending import.
4. Rank candidates by exact fit, active maintenance, clear usage instructions, and low operational risk.
5. For import requests, convert only actual reusable skills. For pure lists, create a concise finder/curator skill instead of copying unrelated repo files.

## Reference

- `references/awesome-agent-skills.md` - curated catalog of official and community agent skills grouped by stack/domain.
